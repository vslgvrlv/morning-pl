# Утренний P&L

Лендинг + демо-дашборд для услуги «построение дашбордов для CEO под ключ».

**Live:** https://vslgvrlv.github.io/morning-pl/

## Файлы

- `index.html` — лендинг
- `demo.html` — интерактивный конструктор-дашборд на вымышленных данных «Демо Детейлинг»

## Стек

Статичный HTML + vanilla JS + Chart.js + Litepicker. Без сборки, без зависимостей — открывается в браузере напрямую.

## Деплой

GitHub Pages из `main`. Изменения на главной → через ~60 сек на live.
